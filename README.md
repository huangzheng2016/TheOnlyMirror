# TheOnlyMirror

既然镜像源太占空间，那为什么不能用反代源呢！

但加速代理域名那么多，为什么不尝试用同一个域名，代理所有的镜像源呢？

原理为不同应用请求镜像源时的UA/PATH/HOST/HEADER会有相应的特征，从而确定请求的镜像源。
